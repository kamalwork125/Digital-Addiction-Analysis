# 📊 Digital Addiction Analysis Dashboard

## 📌 Project Overview
In today’s digital era, excessive use of smartphones and digital platforms has raised serious concerns about **digital addiction**.  
This project analyzes user digital behavior to understand how **screen time impacts sleep patterns and lifestyle** using data analytics tools.

The analysis is performed using **Python, SQL, and Power BI** to extract insights and present them through an interactive dashboard.

---

## 🎯 Objectives
- Analyze daily screen time behavior of users  
- Study the relationship between screen time and sleep duration  
- Identify age groups most affected by high digital usage  
- Build an interactive dashboard for clear visualization of insights  

---

## 📂 Dataset Description
The dataset contains user-level digital usage information such as:
- `user_id` – Unique user identifier  
- `age` – Age of the user  
- `gender` – Gender of the user  
- `screen_time_hours` – Daily screen usage (in hours)  
- `sleep_hours` – Daily sleep duration (in hours)  

The dataset is stored in CSV format and used across all analysis stages.

---

## 🛠 Tools & Technologies
- **Python (Pandas)** – Data loading and preprocessing  
- **SQL (MySQL Workbench)** – Data querying and analysis  
- **Power BI** – Data visualization and dashboard creation  

---

## 🔎 Analysis Performed

### 🐍 Python
- Loaded and explored the dataset  
- Checked data structure and missing values  

### 🗄 SQL
- Calculated total number of users  
- Found average screen time  
- Analyzed screen usage patterns  

### 📊 Power BI
- KPI Cards for key metrics  
- Scatter plot for Screen Time vs Sleep Hours  
- Donut chart for user distribution by age group  
- Clustered column chart for average sleep by screen time
